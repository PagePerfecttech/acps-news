FlipNEWS Storage Setup Guide

Important: You need to set up storage buckets in Supabase before you can upload images and videos to your FlipNEWS application.

Step 1: Go to Supabase Dashboard

Open the Supabase dashboard for your project:

Open Supabase Dashboard

Step 2: Create Required Buckets

You need to create the following buckets:

  • news-images
  • news-videos
  • user-avatars
  • site-assets

For each bucket:

  1. Click the "Create bucket" button
  2. Enter the bucket name exactly as shown above
  3. Make sure "Public bucket" is checked
  4. Click "Create bucket"

Step 3: Set Up Bucket Policies

For each bucket, you need to set up policies to allow public access:

  1. Click on the bucket name
  2. Click on the "Policies" tab
  3. Click "Add policy"
  4. Select "Create a policy from scratch"
  5. Use the following settings:
Setting Value
Policy name Public Access
Allowed operations SELECT, INSERT, UPDATE, DELETE
Target roles anon, authenticated
Policy definition (USING expression) true

Click "Save policy" to create the policy.

Step 4: Test Your Setup

After creating all buckets and policies, return to your FlipNEWS application and try uploading an image or video.

Return to FlipNEWS

Troubleshooting: If you still encounter issues with uploads, check the browser console for error messages. Make sure the bucket names are exactly as specified and that the policies are correctly set up.
